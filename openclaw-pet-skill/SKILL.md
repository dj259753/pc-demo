# 🐧 QQ宠物 Skills 版

> 一只住在你桌面上的像素企鹅，有自己的性格、记忆和情绪。
> 它会观察你的工作状态，在你写代码时陪伴你，偶尔插嘴评论。

## 这是什么？

QQ宠物Skills版是一个基于 OpenClaw 多Agent架构的桌面宠物系统。它会在你的桌面上放一只可互动的像素企鹅，这只企鹅：

- **有独立的人格**：通过独立的 Agent workspace（`qq-pet`）运行，拥有自己的 SOUL.md 和人设，不会干扰你的主 OpenClaw 对话
- **有16种性格**：基于 MBTI 的性格系统，每只企鹅都不一样
- **有记忆**：记住你的习惯、偏好，每天都在了解你
- **有数值系统**：饥饿、清洁、精力三大数值，需要你喂食、洗澡、休息
- **会旁观**：能感知 OpenClaw 主 Agent 的工作状态，偶尔做出有趣的旁观评论
- **会成长**：四维情绪模型（开心/活力/依赖/好奇），和你互动越多越黏人

## 架构

```
┌─────────────────────────────────────────┐
│              你的电脑桌面                   │
│                                          │
│  ┌──────────────┐   ┌──────────────┐     │
│  │ OpenClaw 主窗口│   │ 🐧 宠物窗口   │     │
│  │  你的AI助手   │   │  独立人设     │     │
│  │  自己的人设   │   │  prompt拼装  │     │
│  └──────┬───────┘   └──────┬───────┘     │
│         │                  │              │
│  ┌──────┴───────┐   ┌──────┴───────┐     │
│  │ 主 Agent      │   │ 宠物客户端    │     │
│  │ SOUL.md       │   │ 7层prompt拼装│     │
│  │ 独立对话上下文 │   │ 本地记忆管理  │     │
│  │ 独立记忆      │   │ (不经过Agent) │     │
│  └──────┬───────┘   └──────┬───────┘     │
│         │                  │              │
│         └────────┬─────────┘              │
│                  │                        │
│         ┌────────┴────────┐               │
│         │ OpenClaw Gateway│               │
│         │ openclaw:main   │               │
│         │ (共享LLM推理)   │               │
│         └─────────────────┘               │
└──────────────────────────────────────────┘
```

**核心设计**：宠物客户端自己拼装人设 prompt（性格+心情+记忆+状态），通过 `openclaw:main` 调用共享 LLM 推理，无需注册独立 Agent。企鹅人格完全由客户端控制。

## 安装

Skill 安装后会自动：
1. 创建 `qq-pet` Agent workspace（独立 SOUL.md + IDENTITY.md）
2. 配置宠物专属的16种性格系统
3. 下载桌宠 Electron 应用
4. 设置 Agent 间通信桥接

```bash
# 安装脚本会自动完成以上步骤
bash scripts/install.sh
```

## 使用

安装完成后：
- 宠物窗口会自动出现在桌面右下角
- 单击企鹅 → 轻拍互动
- 双击企鹅 → 打开快捷对话
- 右键企鹅 → 更多功能菜单
- 任务栏 → 查看数值、背包、对话面板

## 技术说明

- **Prompt 驱动人设**：企鹅人格由客户端 7 层动态 prompt 拼装（人设→性格→状态→环境→短期记忆→长期记忆→触发场景），通过 `openclaw:main` 共享 Gateway 推理
- **本地记忆管理**：记忆文件存储在 `~/.openclaw/agents/qq-pet/memory/`，由客户端直接读写，不依赖 Agent 系统
- **降级策略**：OpenClaw 离线时自动切换到本地规则引擎 + 预设台词池
- **旁观机制**：定期读取主 Agent 对话摘要，低概率产生有趣评论
- **AI 配置**：安装时自动从 `~/.openclaw/openclaw.json` 读取 token 生成 `~/.qq-pet/config/ai-config.json`
